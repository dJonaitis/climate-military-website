It is part of an assignment for the Computational Social Science bachelor's programme at the University of Amsterdam.

The source code can be found on the GitHub repository.

The aim of the website is to present a wicked social issue, in a way that is approachable and enticing to navigate through. To this end, I aimed to present the data on the other pages in a way that is easy to read, allowing the reader to draw their own conclusions no matter what their education might be.

This website was made using Flask. All data visualisations are made using the Plotly Python library. All data is managed and stored using SQLite.

The weapons transfer and spending data was taken from SIPRI. The military emission accessibility data was taken from The Military Emissions Gap. Greenhouse gas emission and population data comes from the UN.
